# Kentot TikTok Live Monitor

Node.js + Supabase + TikTok Live Connector

## Cara Jalan

1. Install dependencies:
   ```bash
   npm install
   ```
2. Jalankan aplikasi:
   ```bash
   npm start
   ```

## Environment
- Node.js
- Supabase
- TikTok Live Connector

## Deployment
Pastikan file `package.json` dan `requirements.txt` ada di root project.
